ZJU - SenseTime 3DR Dataset


Introduction

We introduce a new dataset of 3D reconstruction, including the monocular data and large-scale city data captured by mobile device and drone respectively.



Mobile 3D Reconstruction

The mobile 3d reconstruction dataset includes the cases of Scenes and Objects captured by middle-range mobile phones. All the mobile data cases follow dior format. For each case, we also provide a texture model generated by SenseAtlas platform as reference ground-truth (GT).
